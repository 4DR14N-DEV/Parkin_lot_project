# Parking Lot Project - Frontend

## Descripción general

Este proyecto es la interfaz de usuario (Frontend) para un sistema de gestión de parqueaderos. Cuenta con 9 vistas diseñadas para ofrecer una experiencia intuitiva, estructuradas con HTML semántico, estilizadas con CSS y con interactividad dinámica en secciones clave mediante JavaScript.

## Características principales

- **9 Vistas completas:** Navegación fluida entre las diferentes pantallas del sistema.
- **Interactividad:** Lógica implementada en JavaScript para (Creacion dinamica de identificadores unicos, Validacion de credenciales (Usuario) registradas en base de datos, validación de formularios, menús dinámicos, validación de celdas disponibles, inyeccion de HTML dinámico...).
- **Organización modular:** Estructura de archivos limpia separando estructura, estilos y lógica.

## Tecnologías utilizadas

- **HTML5:** Estructura semántica.
- **CSS3:** Estilos, layouts (Flexbox/Grid) y animaciones.
- **JavaScript:** Lógica del lado del cliente.

## Estructura del proyecto

El repositorio está organizado de la siguiente manera:

- `ASSETS/css/` - Archivos de hojas de estilo.
- `ASSETS/js/` - Scripts de interactividad.
- `ASSETS/images/` - Recursos gráficos y logos.
- `ASSETS/fonts/` - Fuentes tipográficas personalizadas.
- `VIEWS/` - Pantallas secundarias de la aplicación.
- `index.html` - Punto de entrada de la aplicación (Login).

## Cómo visualizar el proyecto

Para ver este proyecto en tu máquina local, sigue estos pasos:

1. Clona este repositorio abriendo tu terminal y ejecutando:

   git clone [https://github.com/4DR14N-DEV/Parkin_lot_project.git]

2. Abre la carpeta del proyecto en tu editor de código preferido (ej. VS Code).

3. Abre el archivo index.html en tu navegador web. (Se recomienda usar la extensión Live Server en VS Code para una mejor experiencia).
